## v1.1.0 (2026-05-20)

### 新增功能
- 密码登录鉴权，登录状态保持 24 小时
- 批量导入（多文件上传）
- 批量导入时可选 VLM 自动分析
- 批量移动到指定分类
- 批量设为公共/本群限定
- 单个上传时 VLM 自动识别按钮
- 表情详情显示来源群号
- 表情详情显示哈希值
- 表情详情显示添加时间
- 拉黑功能（删除并加入黑名单）
- 上传进度实时显示

### 改进
- 鉴权方式从 URL token 改为密码登录，更安全
- 表情详情信息更完整

## v1.0.0 (2026-05-20)

### 初始版本
- 独立 Web 服务器，不依赖 AstrBot 内置 WebUI
- 通过引用原插件实例直接操作数据
- 支持查看、编辑、删除表情
- 支持分类筛选和搜索
- 支持单个上传
- 支持 VLM 分析
- 支持分类管理
- Token 鉴权
